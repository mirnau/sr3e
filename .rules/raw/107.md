# Page 107

## Spellcasting Action Declaration

- Casting a spell is declared as a Complex Action.

## Spellcasting Target Number Modifiers

- Situational factors can increase a Sorcery Test target number.
- Cover can increase a Sorcery Test target number.
- The caster's injuries can increase a Sorcery Test target number.
- Each spell currently sustained by the caster adds +2 to the Sorcery Test target number.

## Spell Defense

- Spell Defense dice from a friendly magician can be rolled against a spell.
- Spell Defense successes subtract from the caster's Sorcery Test successes.
