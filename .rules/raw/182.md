## Sorcery Test

- Roll allocated Sorcery dice plus allocated Spell Pool dice for the Sorcery Test.
- The base target number is usually the target's Willpower for mana spells.
- The base target number is usually the target's Body for physical spells.
- The base target number is usually Object Resistance for inanimate objects.
- Elemental Manipulation spells use a base ranged combat target number of 4.

## Rule of One

- If the caster rolls all ones on a Sorcery Test, the target number for the following Drain Resistance Test increases by +2.
