# SR3e Skill Buying Rules (Character Creation)

## Active Skills

- Pool: set by priority (A=50, B=40, C=34, D=30, E=27 points)
- Cost per rating point:
  - Rating **≤ linked Attribute**: 1 SP per point
  - Rating **> linked Attribute**: 2 SP per point
- Maximum base skill rating at creation: **6**
- Unspent points are lost — cannot be saved for later

### Example
Strength 3, want Athletics 6:
- First 3 points (ratings 1–3): 1 SP each = 3 SP
- Next 3 points (ratings 4–6): 2 SP each = 6 SP
- Total: 9 SP

## Knowledge Skills

- Pool: **Intelligence × 5** points (calculated from FINAL Intelligence after attribute spending)
- Linked attribute: **always Intelligence** (predetermined by SR3e rules — player never selects it)
- Cost per rating point:
  - Rating **≤ Intelligence**: 1 SP per point
  - Rating **> Intelligence**: 2 SP per point

## Language Skills

- Pool: **floor(Intelligence × 1.5)** points (calculated from FINAL Intelligence after attribute spending)
- Linked attribute: **always Intelligence** (predetermined by SR3e rules — player never selects it)
- Cost per rating point:
  - Rating **≤ Intelligence**: 1 SP per point
  - Rating **> Intelligence**: 2 SP per point
- Read/Write: automatically set to **floor(spoken rating / 2)** — no extra cost

## Specializations (all skill types)

- Cost: **no extra skill point cost** during character creation
- Mechanic: base skill − 1, specialization = base + 1
- Maximum 1 specialization per skill at creation
- Maximum specialization rating: 7

### Example
Buy Negotiation 6, then specialize → Negotiation 5 (Fast Talking 7)

## Linked Attributes (examples)

| Attribute  | Skills |
|------------|--------|
| Body       | Athletics, Diving |
| Quickness  | Pistols, Stealth, most firearms |
| Strength   | Melee Weapons, Unarmed Combat |
| Intelligence | Computer, Electronics, Biotech |
| Willpower  | Sorcery, Conjuring |
| Charisma   | Etiquette, Negotiation |

The linked attribute acts as a cost threshold only — it does not cap the skill rating.

## Racial Attribute Modifiers Affect Cost Threshold

Different metatypes have different base attribute ratings, which shifts the cost threshold:
- Troll: Body+5, Str+4, Int−2, Cha−2 → physical skills cheaper, mental skills more expensive
- Human: all attributes baseline 6 max
