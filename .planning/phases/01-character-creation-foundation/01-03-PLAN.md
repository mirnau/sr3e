Complete the _replaceHTML method in CharacterCreationApp to mount the Svelte component.

Add import:
```typescript
import CharacterCreationDialog from '@ui/actors/CharacterCreationDialog.svelte';
```

In _replaceHTML method:
```typescript
protected _replaceHTML(_result: unknown, windowContent: HTMLElement, _options: DeepPartial<RenderOptions>): void {
  if (this.#svelteApp) unmount(this.#svelteApp);

  this.#svelteApp = mount(CharacterCreationDialog, {
    target: windowContent,
    props: {
      actor: this.#actor,
      config: CONFIG.sr3e,
      onSubmit: (result: boolean) => {
        this.#wasSubmitted = true;
        this.#onSubmit?.(result);
        this.close();
      },
      onCancel: () => {
        this.#onCancel?.();
        this.close();
      }
    }
  });
}
```

Ensure close() method properly unmounts Svelte app and calls onCancel if not submitted (already in code from Plan 2).
