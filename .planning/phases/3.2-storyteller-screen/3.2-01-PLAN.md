**StorytellerScreenModel.ts** — minimal TypeDataModel, same structure as BroadcasterModel:
```ts
type StorytellerScreenSchema = {
  journalEntryUuid: StringField;
};

export default class StorytellerScreenModel extends foundry.abstract.TypeDataModel<
  StorytellerScreenSchema,
  BaseActor
> {
  static defineSchema(): StorytellerScreenSchema {
    return {
      journalEntryUuid: new foundry.data.fields.StringField({
        required: false,
        initial: "",
      }),
    };
  }
}
```

**StorytellerScreenSheet.ts** — extends SR3EActorBase, follows BroadcasterSheet exactly:
- `DEFAULT_OPTIONS`: id with `randomID()`, classes `["sr3e", "sheet", "actor", "storytellerscreen", "ActorSheetV2"]`, position `{ width: 900, height: "auto" }`, window `{ resizable: true }`, tag `"form"`, submitOnChange: true, closeOnSubmit: false
- `_replaceHTML()`: unmountAllApps, mount StorytellerScreenApp into windowContent with props `{ actor: this.document as Actor }`; push to `this.apps`; call `this._injectFooter(form)`
- `_injectFooter()`: copy from BroadcasterSheet (insert `.actor-footer` div before resize handle)
- `_tearDown()`: unmountAllApps, call super
- `_onSubmit()`: return false
- `_onDrop()`: preventDefault, stopPropagation, return false

**Props:** `{ actor: Actor }` — required, no default.

**Structure:**
```svelte
<script lang="ts">
  import KarmaManager from "./storyteller-screen/KarmaManager.svelte";

  let { actor }: { actor: Actor } = $props();
</script>

<div class="storyteller-screen">
  <div class="storyteller-screen__panel">
    <KarmaManager {actor} />
  </div>
</div>
```
